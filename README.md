## 🏆 Featured Projects

### 🍳 [ChefSpeak](https://github.com/dhanunjay1729/chefspeak)
> **An intelligent, multilingual, voice-guided cooking companion.**
> Built a full-stack, voice-activated assistant using **React 19**, **Node.js**, **Firebase**, and **OpenAI GPT-4**. Features real-time multilingual voice guidance, smart timers, and in-browser wake word detection using TensorFlow.js.

### 🌍 [Where Have I Been](https://github.com/dhanunjay1729/where-have-i-been)
> **A privacy-first, 3D interactive map experience for location history.**
> Architected a client-side **Next.js** application using **Mapbox GL JS** to visualize large-scale travel data (GPX, KML, JSON). Features 3D terrain, gradient routes, and a beautiful cyberpunk aesthetic. 100% private with no server uploads.

### 🔗 [Distributed P2P Ledger](https://github.com/dhanunjay1729/Distributed-P2P-Ledger)
> **A decentralized blockchain ledger implementing custom gossip and consensus.**
> Engineered a P2P network in **Go (Golang)** and deployed via **Docker Compose**. Implemented a custom HTTP gossip protocol, turn-based consensus utilizing cryptographic hashing, and a "Longest Chain Wins" conflict resolution strategy.

### 🧠 [Spaced Repetition App](https://github.com/dhanunjay1729/spaced-repetition-app)
> **AI-enhanced flashcard tool optimized by the SM-2 algorithm.**
> A modern study tool built with **React**, **Zustand**, and **Firebase** that calculates optimal study intervals and uses the **OpenAI API** to generate contextual hints and extract text directly from images.

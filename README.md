## 🏆 Featured Projects

### 🍳 [ChefSpeak](https://github.com/dhanunjay1729/chefspeak)
> **An intelligent, multilingual, voice-guided cooking companion.**
> Built a full-stack, voice-activated assistant using **React 19**, **Node.js**, **Firebase**, and **OpenAI GPT-4**. Features real-time multilingual voice guidance, smart timers, and in-browser wake word detection using TensorFlow.js.

### 🌍 [Where Have I Been](https://github.com/dhanunjay1729/where-have-i-been)
> **A privacy-first, 3D interactive map experience for location history.**
> Architected a client-side **Next.js** application using **Mapbox GL JS** to visualize large-scale travel data (GPX, KML, JSON). Features 3D terrain, gradient routes, and a beautiful cyberpunk aesthetic. 100% private with no server uploads.
